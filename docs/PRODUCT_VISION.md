## 1. 한 줄 정의

**주식시장의 틴더 — 캐주얼 투자 발견 앱.**
종목을 스와이프하며 "내 취향의 종목"을 발견하고, 뜨기 전의 신호(💎)를 남보다 먼저 만나는 곳.

- 분석 앱(머니터링·증권사 리포트)이 "이 종목이 어떤 종목인가"를 **설명**한다면,
- FOMO Club은 "**너**라면 어떤 종목에 멈춰 설지"를 **안다**.
- 우리는 정답(사라/팔아라)을 주지 않는다. **발견의 즐거움과 취향의 자각**을 준다.

> **머니터링은 종목을 분석해주고, FOMO Club은 너를 안다.**

---

## 2. 핵심 철학

### 2.1 정답이 아니라 발견
주식 앱은 "수익률"을 약속하다 신뢰를 잃는다. 우리는 약속하지 않는다.
대신 데이팅 앱의 문법을 빌린다 — **가볍게 넘기고(swipe), 끌리면 멈추고(♥), 더 알아가는(depth)** 경험. 매수 버튼이 종착지가 아니다. **"이 종목 흥미롭네"가 종착지**다.

### 2.2 취향은 자각하게 만드는 것
사람은 자기 투자 취향을 모른다. 무엇에 멈춰 서는지, 어떤 섹터에 반복해 끌리는지 — 스와이프 행동이 그걸 **거울처럼 비춘다**. 우리는 그걸 사실로 돌려준다: "너는 이런 종목에서 자주 멈췄어." 판정이 아니라 패턴.

### 2.3 FOMO는 적이 아니라 연료
"남들 다 타는데 나만 놓쳤나"는 불안은 끄는 대상이 아니라 **발견의 입구**다. 포모 점수는 그 쏠림을 정직하게 보여주는 온도계 — 단, 쏠림을 **추종하라고** 말하지 않는다. "지금 여기 사람이 모인다"는 사실까지만.

### 2.4 조기 발견이 진짜 가치 (💎)
남들 다 아는 종목을 늦게 보여주는 건 가치가 없다. **뜨기 전에, 수급이 먼저 들어오는 순간**을 짚어주는 것 — 그게 틴더가 "아직 아무도 모르는 매력적인 상대"를 먼저 보여주는 것과 같다. (단 §5의 예측 금지선을 절대 넘지 않는다.)

## 3. 포지셔닝 & 두 표면, 한 엔진

### 3.1 데이팅 문법 차용
| 데이팅 앱 | FOMO Club |
|---|---|
| 프로필 카드 스와이프 | **종목 카드 스와이프** |
| ♥ / pass | 관심 / 넘김 |
| 매칭 | 취향 유사도("네가 멈춰보던 패턴과 닮음") |
| 뜨기 전 좋은 상대 | 💎 조기 발견(수급 선행 신호) |

분석 앱(머니터링·증권사 HTS)은 "종목이 어떤지"를 **모두에게 똑같이** 설명한다. FOMO Club은 **너의 스와이프**로 너의 취향을 배우고, 너에게 맞는 발견을 준다. 같은 종목이라도 우리는 "너라면 멈출 종목인가"를 본다.

### 3.2 두 표면, 한 엔진 (v5 핵심)
제품은 **두 개의 표면**을 갖지만 **하나의 엔진**으로 돈다.

```
        ┌─────────────────────────────────────────────┐
        │             한 엔진: 주목 엔진                │
        │      포모 점수(쏠림 온도) + 💎(조기 발견)      │
        └───────────────┬─────────────────┬───────────┘
                        │                 │
            ┌───────────▼──────┐  ┌───────▼────────────┐
            │ ① 발견 (심장)     │  │ ② 콘텐츠            │
            │ 종목 카드 스와이프 │  │ 뉴스·브리핑 읽기     │
            │ 종목 전용         │  │ (별도 표면)         │
            └──────────────────┘  └────────────────────┘
```

- **① 발견 표면 = 제품의 심장.** 종목 카드를 스와이프한다. **종목 전용**이다.
- **② 콘텐츠 표면 = 읽는 곳.** 뉴스·브리핑·내러티브를 소비한다. 발견을 거들지만 발견을 대체하지 않는다.
- **두 표면은 같은 엔진**(포모 점수 + 💎)을 공유한다. 콘텐츠에서 회자되는 것이 발견 신호로, 발견에서 끌린 것이 콘텐츠 읽기로 이어진다.

> ⚠️ **폐기**: v4의 "7종 카드(종목·테마·매크로·이벤트·내러티브·인물·…)를 한 피드에 비율로 섞는다"는 더 이상 정본이 아니다. **발견 표면은 종목 카드만.** 테마·매크로·이벤트·내러티브는 **콘텐츠 표면**의 재료이거나 종목 카드 안의 맥락(사실 한 줄)으로만 쓴다.

### 3.3 사용자 여정
```
포모 점수로 "지금 쏠린 곳" 인지  →  종목 카드 스와이프(발견)
   →  끌리면 ♥ · depth 로 더 알아봄  →  취향이 쌓임(거울)
   →  💎 로 뜨기 전 종목을 남보다 먼저  →  다시 옴(재방문)
콘텐츠 표면(뉴스·브리핑)은 이 루프에 맥락과 회자를 공급.
```

### 3.4 콜드스타트
새 유저는 취향 데이터가 0 → 처음엔 개인화할 게 없다. **초기엔 포모 점수(쏠림)·💎 기반 발견으로 시작 → 스와이프가 쌓이며 점점 취향 매칭**으로 정밀화. "첫날 텅 빈 개인화"를 만나지 않게.

## 4. 발견 표면 — 종목 카드 (제품의 심장)

### 4.1 카드 = 종목 전용
발견 표면의 카드는 **종목 카드 한 종류**다. 한 장에:
- 종목명·티커, **포모 점수**(쏠림 온도, §6),
- 💎 배지(해당 시 — 조기 발견 신호, §5),
- 끌림을 만드는 사실 한 줄(예: "외국인 3일 연속 순매수 · 6/20 장마감"),
- 넘김 / ♥(관심) / depth(상세).

### 4.2 카드 안의 맥락은 "사실 한 줄"까지만
테마·매크로·수급·TA(기술적 지표)는 **카드 안의 보조 사실**로만 등장한다. 독립 카드로 피드를 채우지 않는다.
- ✅ "반도체 섹터 쏠림 · 외국인 순매수" (맥락 한 줄)
- ❌ 테마 카드·매크로 카드를 종목 카드와 같은 덱에 섞기

### 4.3 콘텐츠 표면의 구조 (구현 후속)
콘텐츠 표면(② 읽기)은 **별도 표면**으로 둔다. 구성 요소(우선순위 아님, 구조만):
- 오늘의 시장 브리핑(쏠림 요약), 테마/내러티브 읽을거리, 이벤트·일정 맥락.
- 발견 표면과 같은 엔진(포모 점수·💎)으로 정렬·강조.
- **신설 문서(DISCOVERY_OS 등)는 보류** — 이 정본에 구조만 남기고, 발견 척추(§11)가 선 다음 구체화한다.

## 5. 💎 조기 발견 (v5 패치 — 철학 레벨)

💎는 **"뜨기 전" 신호**다. 수급(외국인·기관·개인 순매매)이 **먼저 들어오는 중**임을 사실로 표시한다.
- confidence는 낮고(L), 잠재 가치(leverage)는 높은 영역 — 그래서 "남보다 먼저"의 가치가 거기 있다.
- 토대: 첫 감지일·연속 순매수일 등 **시점 박힌 사실**(`KEYWORD_ENGINE_SPEC`·`DATA_ENGINE_STRATEGY` 와 정합).

### 5.1 예측 금지선 (절대)
💎는 **예측이 아니다.** 표현은 사실까지만:
- ✅ "외국인이 3일 연속 순매수하며 수급이 먼저 들어오는 중 (6/20 장마감 기준)"
- ❌ "곧 오른다 / 지금 안 사면 늦는다 / 급등 임박" — **전부 금지.**
- 💎는 "남들이 아직 안 본 곳에 자금 흐름이 보인다"는 **관찰**이지 매수 신호가 아니다.

## 6. 포모 점수 — 헤리티지이자 연료 (v5 패치)

포모 점수는 우리의 **헤리티지**(브랜드 자산)다. 하지만 v5에서 그 역할을 못 박는다:

> 포모 점수는 **발견의 연료**지 **진열 상품**이 아니다.

- ✅ 종목 카드 안에서 "지금 얼마나 쏠렸나"를 보여주는 온도계 → 스와이프할 이유를 만든다.
- ❌ **포모 점수 랭킹표를 독립 화면으로 진열 금지.** 점수 자체를 구경하는 제품이 아니다.
- TA(기술적 지표)도 동일 — **카드 안 사실 한 줄**까지만. TA 차트·지표 진열 화면을 따로 만들지 않는다.

점수는 발견(스와이프)으로 사람을 끌어들이는 수단이고, 목적은 어디까지나 **종목 발견과 취향 자각**이다.

## 7. 매칭 — 취향 유사도 사실까지만 (v4 §13 → v5 수정)

매칭은 데이팅의 문법이지만 **투자 추천으로 미끄러지면 안 된다.**

- ✅ "**네가 멈춰보던 패턴과 닮은** 종목이야" — 행동 데이터 기반 취향 유사도(사실).
- ❌ "**사도 되는 자리** / 너한테 맞는 매수 타이밍" — 금지.
- 매칭의 의미는 "비슷한 결의 종목"이지 "사야 할 종목"이 아니다. 판단은 언제나 유저.

---

## 8. BM — 확정 보류 + 발굴 성적표 (v4 §17 → v5 수정)

수익 모델은 **확정 보류**한다(지금 BM에 제품을 맞추지 않는다). 대신 가치 가설을 사실로 검증할 **발굴 성적표**를 먼저 만든다.

### 8.1 발굴 성적표
- 내가 ♥한 종목·내가 본 💎가 **그 뒤 어떻게 됐는지**를 사실로 되돌려준다.
- 예: "네가 4/2에 본 💎 종목, 그 뒤 외국인 순매수 9일 연속." (사실·시점만, 수익률 단정·자랑 금지)
- 목적: **재방문(retention)** 과 **지불 의향(WTP)** 의 근거를 데이터로 확인 → BM은 그 다음에 결정.

## 9. UX 원칙 · 정보구조 · 핵심 기능

### 9.1 UX 원칙
- **가볍게**: 한 손 스와이프. 학습 곡선 0. 틀려도 부담 없는 넘김.
- **정직하게**: 출처·시점·confidence를 항상 함께. 데이터 없으면 "아직 없음"(가짜 금지).
- **판정하지 않게**: 강세/약세 양면 제시, 명령형·미래 단정·추천 금지. 재료를 주고 판단은 유저.

### 9.2 정보구조 (현재 기준)
- **발견 탭(심장)**: 종목 카드 스와이프 → depth 상세.
- **콘텐츠 탭(읽기)**: 브리핑·뉴스(별도 표면, 후속 구체화).
- **나의 발굴(성적표)**: ♥·💎 히스토리와 그 후 사실.

### 9.3 핵심 기능 (발견 척추 기준)
종목 카드(포모 점수+💎) → depth 상세(사실·출처·양면) → 정렬·필터 → 개인화(취향 유사도) → 발굴 성적표.

## 10. 규제 · 안전 · 네이밍

### 10.1 절대 원칙 (위반 시 자동 폐기)
- **매수·매도·목표가·수익률·예측·판정 금지.** 명령형("사라")·미래 단정("오른다")·추천("추천")·신호("매수신호") 전부.
- **출처·시점·confidence 필수.** 모든 사실에 source·기준일·신뢰도.
- **강세·약세 양면.** 한쪽만 살아남으면 그 사실을 정직하게(stanceNote).
- **source-kind separation.** 뉴스/공식=분석 근거, 커뮤니티=워딩(여론)만. 섞지 않음.
- **"추천"이 아니라 "취향 매칭·발견·판단 재료".**
- **정직한 숫자.** 가짜·환각 데이터 금지. 원문에 grounded된 것만.

### 10.2 네이밍
- 제품: **FOMO Club**. 한 줄: "주식시장의 틴더".
- 대외 카피는 "발견·취향"을 말하고 "수익·추천"을 말하지 않는다.

## 11. 빌드 순서 (v5)

발견 표면(심장)부터 척추를 세우고, 콘텐츠 표면과 BM 실험은 그 뒤.

1. **발견 척추**
   1) 포모 점수가 박힌 **종목 카드** 스와이프
   2) **depth 상세**(사실·출처·시점·양면)
   3) **정렬·필터**(쏠림순·💎순 등)
   4) **튜닝 + TA**(카드 안 사실 한 줄 수준)
   5) **개인화**(스와이프 행동 → 취향 유사도 매칭)
   6) **발굴 성적표**(♥·💎 그 후 사실 → retention/WTP 근거)
2. **콘텐츠 표면**(② 읽기 — 브리핑·뉴스, 같은 엔진으로 정렬)
3. **BM 실험**(발굴 성적표 데이터가 가설을 지지하면)

> 옛 Phase 1~4(데이터 엔진)는 발견 척추의 **연료**다 — `DATA_ENGINE_STRATEGY` 의 수집→이해→재가공→응축이 종목 카드의 사실을 공급한다. 엔진과 발견 표면은 대립이 아니라 같은 엔진의 두 면.

## 12. 에이전트 협업 기준 (v4 §21 + v5)

에이전트(자동 개발 군단)는 이 정본을 매일 읽고 따른다. **다음 제안은 자동 거부**한다:

- 발견 표면(종목 카드 덱)에 **테마·매크로·이벤트 카드를 섞는** 제안 (→ 콘텐츠 표면 또는 카드 안 한 줄로).
- **포모 점수/TA 랭킹·차트를 독립 화면으로 진열**하는 제안 (점수는 연료지 진열 상품 아님).
- 💎·점수에 **예측·매수신호·목표가**를 붙이는 제안 (사실까지만).
- 매칭을 "**사도 되는 자리**"로 표현하는 제안 (취향 유사도 사실까지만).
- 마스코트 주인공화·감정 진정·투표·캘린더 등 **옛 정체성**으로 회귀하는 제안.
- BM에 제품을 맞추는 제안 (BM 보류, 발굴 성적표 먼저).

에이전트는 변경 시 **순수 함수 + 테스트**, PR 단위, CI 초록, 출처·시점·confidence 원칙을 지킨다.

## 최종 선언

FOMO Club은 **두 표면(① 종목 발견 스와이프 · ② 콘텐츠 읽기)** 을 **하나의 주목 엔진(포모 점수 + 💎 조기 발견)** 으로 굴리는 **주식시장의 틴더**다.
우리는 정답을 팔지 않는다. 발견의 즐거움, 취향의 자각, 그리고 남보다 먼저 보는 눈(💎)을 준다.

> **머니터링은 종목을 분석해주고, FOMO Club은 너를 안다.**
